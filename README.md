# Yolov8

## Description

This is a project that aims to detect object.

## Installation

To install this project, follow these steps:

1. Install
```
pip install -e .
```

## Usage

To use this project, follow these instructions:

1. train
```
python train.py
```
2. test
```
python test.py
```
3. detect 
RGB detect 
```
python detectRGB.py
```
IR detect 
```
python detectIR.py
```
4. export
```
python export.py
```
5. split(split by  grayscale)
```
python split.py
```
